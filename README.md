# IT Expenditure Analysis Dashboard
## Overview

This project presents a Power BI dashboard that analyses IT expenditure across different departments, categories, and time periods.

The goal is to identify spending patterns, optimise costs, and support better financial decision-making.

## Objectives
* Analyse IT spending across departments
* Identify high-cost areas
* Track expenditure trends over time
* Enable interactive filtering and exploration

## Dashboard Features
* Total IT spending KPI
* Spending trends over time
* Department-wise expenditure breakdown
* Category-wise cost analysis (hardware, software, services, etc.)
* Interactive filters (time, department, category)

## Key Insights
* Certain departments contribute the majority of IT spending
* IT costs show trends over time (e.g., growth or seasonal spikes)
* A few categories (e.g., software or infrastructure) dominate expenses
* Opportunities exist to optimise high-cost areas

## Business Impact

This dashboard helps organisations:

* Monitor and control IT budgets
* Identify cost-saving opportunities
* Improve resource allocation
* Support strategic decision-making
